# WebCTF

The WebCTF is open-sourced software licensed under the [MIT license](https://opensource.org/licenses/MIT).


## Setup
- `git clone https://github.com/BISDAKPOWERS/WebCTF.git`
- `composer install`
- `npm install`
- `cp .env.example .env`
- `php artisan migrate:refresh && php artisan db:seed`
